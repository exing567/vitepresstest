## 📖 欢迎使用 Eric's Notes
这是一个专注于学术笔记整理的静态网站，目前包含**微积分**和**历史**两大核心学科：

### ✨ 核心功能
- 📝 结构化笔记：按学科/单元分层，知识点清晰易查
- 🧮 LaTeX 公式：微积分笔记内置完整的数学公式渲染，符合学术排版规范
- 🔍 全文搜索：支持关键词检索，快速定位所需内容
- 🚀 极速加载：静态站点构建，访问速度快，无后端依赖

### 📌 使用指引
1. 点击侧边栏/首页按钮进入对应学科笔记；
2. 右上角搜索框可输入关键词检索全文；
3. 笔记内公式支持复制，例题可直接参考解题思路；
4. 如需本地部署，可克隆源 [GitHub](https://github.com/vuejs/vitepress) 仓库自行构建。
